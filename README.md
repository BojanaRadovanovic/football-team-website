# football-team-website

This project is a responsive website dedicated to the Serbia national football team, developed using Bootstrap.

## Features
- Responsive design for different screen sizes
- Sections for players, matches, and team information
- Clean and structured layout using Bootstrap components

## Technologies
- CSS
- Bootstrap

## Description
The website presents information about the Serbia national football team, including players, match details, and general team overview.
